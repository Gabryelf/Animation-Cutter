  # PixelForge — Анимация гуманоида из спрайт-листа
</div>

**PixelForge** — это веб-приложение для создания покадровой анимации персонажей (гуманоидов) из пиксель-арт спрайт-листов. Позволяет загрузить PNG-изображение с сеткой кадров, вручную собрать последовательность анимации и просмотреть результат с регулируемой частотой кадров.

## 📋 Возможности

- **Загрузка спрайт-листа** — перетаскивание или выбор PNG-файла.
- **Нарезка на кадры** — задание количества столбцов и строк для автоматического разбиения изображения.
- **Галерея кадров** — просмотр всех вырезанных кадров с возможностью добавлять их в анимацию одним кликом.
- **Редактирование последовательности**:
  - Добавление всех кадров сразу.
  - Удаление/очистка всей последовательности.
  - Изменение порядка кадров (кнопки вверх/вниз) и удаление отдельных кадров.
- **Предпросмотр анимации**:
  - Воспроизведение / остановка.
  - Регулировка FPS (3–30 кадров в секунду).
  - Пиксель-арт рендеринг с сохранением пропорций.
- **Адаптивный интерфейс** — тёмная тема, стеклянная панель, поддержка мобильных устройств.

## 🚀 Как использовать

1. **Загрузите спрайт-лист**  
   Нажмите на область загрузки или перетащите PNG-файл с пиксель-артом. Изображение должно содержать сетку кадров (например, 4 столбца × 1 строка для походки).

2. **Настройте разбиение**  
   Укажите количество **столбцов** (кадров по горизонтали) и **строк** (кадров по вертикали), затем нажмите «Разрезать на кадры».

3. **Соберите анимацию**  
   - В галерее кликните на любой кадр — он добавится в последовательность.  
   - Используйте кнопку «Добавить все кадры» для быстрого заполнения.  
   - В панели последовательности меняйте порядок кадров стрелками или удаляйте ненужные.

4. **Настройте воспроизведение**  
   - Ползунком **FPS** задайте скорость анимации.  
   - Нажмите «Воспроизвести» для предпросмотра.  
   - «Стоп» останавливает проигрывание.

5. **Очистка**  
   Кнопки «Очистить анимацию» удаляют все кадры из последовательности, не затрагивая галерею.

---

## 🛠 Требования к спрайт-листу

- Формат: **PNG** (другие форматы не поддерживаются).
- Сетка кадров должна быть равномерной (все кадры одного размера).  
  *Рекомендация*: ширина и высота изображения должны делиться на количество столбцов и строк без остатка — тогда кадры будут целыми пикселями.
- Пример: спрайт 256×64 px при 4 столбцах и 1 строке даст кадры 64×64 px.
